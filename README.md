# Warehouse Locator v2.0

Aplikasi pencarian lokasi material berbasis GitHub Pages.

## Fitur

- Search Material
- Search Description
- Search Storage Bin
- Dashboard
- Copy Storage Bin
- Dark Mode
- Responsive
- Progressive Web App (PWA)

## Cara Update Data

1. Ganti file data.json
2. Commit ke GitHub
3. Refresh browser

Developed for SAP Warehouse.
